How to download files from GitHub repo

git clone 

syntax : 

git clone repo_url

git clone https://github.com/prasannatrainer80/JbTrg

How to add the files to staging area

git add .

How to add the files to local repo

git commit -m "message"

How to add the files to remote repo

git push origin main

git diff -> Used to find the difference between old commit (vs) staging

git status -> Need to find out the current status of GitHub repo
